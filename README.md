# BTS ARMY World ✦

> 一个为中文ARMY打造的防弹少年团粉丝互动网站

**🌐 在线访问 → [simsim-kk.github.io/bts-army-world](https://simsim-kk.github.io/bts-army-world/)**

---

## ✨ 功能一览

### 🎴 抽签故事
翻牌决定七个人的角色，AI生成专属的7v1乙女向命运故事。
还有社区问题包——校园、职场、古风、女尊……ARMY自己设计剧本，大家一起玩。

### ✍️ 专属故事
选择心情、故事类型、文风、成员，AI写一个只属于你的故事。
散文诗 / 温柔日常 / 偶像剧 / 克制深情 / 随机，五种风格可切换。

### 🌙 防弹梦境档案馆
梦到防弹了？记录下来和大家分享。
可以按成员标签筛选、点赞、用专属编号查看自己的所有梦境。

### 🎵 歌词卡片
输入喜欢的歌词，生成四种风格的精美卡片，可保存分享。
下方有流动的歌词墙，ARMY可投稿自己喜欢的歌词。

### 💜 今日陪伴
每天抽一位member陪伴你，附专属运势和标签，生成可分享的运势图。

### 🎖️ 入团纪念日
输入入团日期，生成专属纪念证书。

### ✦ 防弹语录
随机抽取防弹语录，ARMY可投稿喜欢的语录，审核后进入语录池。

### 🧬 MBTI灵魂配对
输入你的MBTI，查看与七位成员的契合度排行。

### 🔮 八字命盘匹配
输入生辰八字，推算五行命格与成员的缘分。

### 📖 成员图鉴
七位成员的星座、生肖、MBTI、性格深度解析。

### 🎧 心情歌单
十种心情各对应八首精选歌曲，一键复制到网易云。

### 💌 留言板
写下你想说的话，留在这里。
